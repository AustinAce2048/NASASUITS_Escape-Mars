get

We want a pinch up and down gesture to be done while an object is focused.

We don't want someone to pinch, unfocus an object, then refocus before doing pinch up. We also want to avoid focusing a different interactable during this process. While the latter is difficult to do since a person might focus nothing before focusing on another interactable, it's theoretically possible.
