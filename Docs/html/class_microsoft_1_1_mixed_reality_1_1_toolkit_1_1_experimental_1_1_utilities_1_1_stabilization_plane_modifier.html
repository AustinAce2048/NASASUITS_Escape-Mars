Detailed Description

StabilizationPlaneModifier handles the setting of the stabilization plane in several different modes.

It does this via handling the platform call to HolographicPlatformSettings::SetFocusPointForFrame Using StabilizationPlaneModifier will override DepthLSR. This is automatically enabled via the depth buffer sharing in Unity build settings StabilizationPlaneModifier is recommended for HoloLens 1, can be used for HoloLens 2, and does a no op for WMR
