void Microsoft.MixedReality.Toolkit.Examples.Demos.EyeTracking.BlendOut.Disengage ( )

Once the user looks away from the hologram, determine whether the user has dwelled at it before.

If yes, start blending it out and destroy it if that option was selected. If not, we can still blend it out a little, but keep it alive and visible.
