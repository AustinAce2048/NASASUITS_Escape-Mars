Detailed Description

Implements a movement logic that uses the model of angular rotations along a sphere whose radius varies.

The angle to move by is computed by looking at how much the hand changes relative to a pivot point (slightly below and in front of the head).

Usage: When a manipulation starts, call Setup. Call Update any time to update the move logic and get a new rotation for the object.
